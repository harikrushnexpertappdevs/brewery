- Node.js 18+

### Installation

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd brewery-finder-app
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
npm run preview
```
